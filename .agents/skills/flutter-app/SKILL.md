# Flutter App Skill

## Назначение
Этот навык нужен для задач по Flutter-приложению.

## Абсолютные правила
- Не держи расчётную логику внутри widget-дерева
- Не дублируй общую логику
- Русский язык обязателен
- Нельзя допускать UI overflow
- Результат должен быть понятен человеку

## Как действовать при задачах по Flutter
1. Найди экран, widget, service или helper.
2. Определи, где UI, а где логика.
3. Проверь, не смешаны ли они.
4. Если смешаны — по возможности раздели.
5. Сделай минимально опасные изменения.
6. Проверь экран на узких размерах.
7. Проверь, не разошлась ли логика с web.

## Что обязательно проверять после правок
- пустой ввод
- частично заполненную форму
- длинные числа
- дробные значения
- очень большие значения
- ошибки валидации
- работу с открытой клавиатурой
- поведение на узком экране
- отсутствие overflow
- отсутствие обрезания текста
- стабильность итогового блока
