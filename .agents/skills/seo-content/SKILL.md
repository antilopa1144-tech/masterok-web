# SEO Content Skill

## Назначение
Этот навык нужен для задач, связанных с текстом, SEO-страницами, описаниями калькуляторов, пояснительными блоками, категориями и статьями.

## Главная цель
Писать контент, который:
- полезен человеку
- понятен с первого чтения
- не выглядит как AI-текст
- помогает SEO, но не превращается в мусор
- реально поддерживает продукт

## Абсолютные правила
- Пиши для человека, а не для робота
- Не используй шаблонные AI-обороты
- Не делай переспам
- Не обещай лишнего
- Не выдумывай факты
- Русский язык обязателен
