# Обратная связь с сайта → Telegram

Виджет «💬 Отзыв» (плавающая кнопка слева снизу + ссылка в футере) отправляет
сообщения пользователей прямо вам в Telegram. Без базы данных — на Timeweb
Cloud Apps файловая система эфемерна, поэтому доставка «наружу» в личку и есть
хранилище отзывов.

## Что настроить (5 минут)

1. **Создать бота.** В Telegram напишите [@BotFather](https://t.me/BotFather) →
   `/newbot` → задайте имя и username. BotFather пришлёт **токен** вида
   `1234567890:AAderf...`. Это `TELEGRAM_BOT_TOKEN`.

2. **Узнать свой chat_id.** Напишите боту [@userinfobot](https://t.me/userinfobot)
   `/start` — он пришлёт ваш числовой `Id`. Это `TELEGRAM_FEEDBACK_CHAT_ID`.
   - Хотите получать отзывы в группу/канал — добавьте своего бота туда и
     возьмите chat_id группы (у групп он отрицательный, напр. `-100123...`).

3. **Активировать бота.** Откройте чат со своим ботом и нажмите **Start** —
   иначе Telegram не даст боту написать вам первым.

4. **Прописать env на Timeweb** (панель Cloud Apps → переменные окружения):
   ```
   TELEGRAM_BOT_TOKEN=1234567890:AAderf...
   TELEGRAM_FEEDBACK_CHAT_ID=123456789
   ```
   Передеплоить приложение, чтобы переменные подхватились.

Готово. Отзывы будут приходить так:
```
💬 Отзыв с сайта Мастерок
👍 Нравится

Очень удобный калькулятор плитки, спасибо!

📨 Контакт: @user
🔗 Страница: Калькулятор плитки — Мастерок https://getmasterok.ru/...
🖥 390×844 · bronze
```

## Если не настроено

Сайт не падает: при отсутствии env отзыв не теряется молча, а пишется в
логи сервера с пометкой `[feedback] Telegram не настроен`. Пользователь
всё равно видит «Спасибо! Получил».

## Где что в коде

| Файл | Назначение |
|------|------------|
| `src/components/feedback/FeedbackWidget.tsx` | Виджет (кнопка + чат-панель) |
| `src/components/feedback/FeedbackFooterLink.tsx` | Ссылка в футере, открывает виджет |
| `src/app/api/feedback/route.ts` | Приём: валидация, rate-limit (5/мин на IP), honeypot |
| `src/lib/feedback/telegram.ts` | Форматирование и отправка в Telegram Bot API |

## Что прикрепляется к отзыву (контекст)

URL и заголовок страницы, размер экрана, тема оформления. Имя/почта — только
если пользователь сам указал контакт. IP не передаётся в Telegram (используется
лишь на сервере для rate-limit).

## Идеи на будущее

- **Двусторонний диалог**: ответ из Telegram → обратно в виджет на сайте.
  Требует постоянного хранилища сессий и вебхука бота — отдельная задача.
- Дубль отзывов в Langfuse/Google Sheet как архив.
- Антиспам посильнее (hCaptcha) — если пойдёт волна ботов.
