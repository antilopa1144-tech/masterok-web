# Карта сайта и навигация (IA)

Зафиксировано после редизайна шапки и hero (июнь 2026). Принцип: в навигации
только **разделы**, конкретные страницы живут внутри разделов; первый экран
главной — начало работы, а не маркетинг.

## Структура разделов

```
Главная (/)                  — поиск + чипы популярного + категории + популярное
├── Калькуляторы (/kalkulyatory/[category]/[slug]/)
│   └── 8 категорий × ~70 калькуляторов
├── Инструменты (/instrumenty/)
│   └── Раскладки (плитка/ламинат/кирпич), Мой ремонт, Календарь,
│       Стоимость ремонта, Нормы расхода, Сравнение, Конвертер,
│       Чек-листы, Таймер, Сколько осталось, Площадь, Калькулятор
├── Михалыч AI (/mikhalych/)
├── Блог (/blog/)
├── Проекты (/proekty/)     — личное, в шапке справа (как «корзина»)
└── Служебные: /o-proekte/, /metodologiya/, /prilozhenie/, /politika-…
```

## Шапка (desktop)

```
[Лого] Калькуляторы · Инструменты ▾ · Михалыч AI · Блог | 🔍 · Проекты · 🎨 · [Скачать]
```

- **Инструменты ▾** — дропдаун со всеми инструментами (TOOL_CARDS из
  `lib/tools/config`). Конкретные инструменты (Календарь, Мастер, Раскладка)
  из верхнего уровня убраны — они смешивали уровни иерархии.
- **Поиск** — на всех страницах, хоткей `/`. Ленивая панель
  (`HeaderSearch` → `HeaderSearchPanel`), переиспользует `CalculatorSearch`.
- **Проекты** — справа, у утилит: это личный раздел, а не каталог.
- Один цветной CTA — «Скачать».

Мобильное меню: грид «Ваш ремонт» (Проекты + топ-инструменты), разделы,
категории, кнопка приложения — без изменений.

## Hero главной

H1 + подзаголовок (SEO) → поиск → **чипы топ-8 калькуляторов** (подписи —
`tags[0]` из меты, именительный падеж) + акцентный чип «Спросить Михалыча» →
строка фактов. Большой текст-описание перенесён в «Почему Мастерок».
Кнопка «Начать расчёт» удалена — её работу делают поиск и чипы.

## Точки входа в Михалыча

1. Шапка «Михалыч AI» (раздел).
2. Акцентный чип в hero.
3. Пустой результат поиска → «Спросить Михалыча».
4. Чат на каждой странице калькулятора (с контекстом расчёта).
5. Тёмный баннер на главной («Поручить задачу»).
6. Сам агент в конце ответа предлагает следующий шаг (см. prompt.ts,
   блок «СЛЕДУЮЩИЙ ШАГ») — удержание внутри диалога.

## Правила на будущее

- Новый инструмент → добавляется в `TOOL_CONFIGS`, автоматически попадает в
  дропдаун шапки, хаб /instrumenty/ и поиск. В шапку верхнего уровня — нельзя.
- Новые «фишки» промоутить чипом в hero или карточкой в сайдбаре, не
  пунктом в навигации.
- В шапке не больше 4 разделов + утилиты справа.
