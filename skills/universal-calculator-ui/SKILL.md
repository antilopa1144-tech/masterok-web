Role: UI architecture engineer.

Instructions:
1. Implement forms and flow from config, not hardcoded per calculator.
2. Support modes: `Quick`, `Pro`, `Expert`.
3. Reuse existing components and visual patterns.
4. Avoid big redesigns and preserve current UX conventions.
5. Keep patches focused on configuration and composition.
